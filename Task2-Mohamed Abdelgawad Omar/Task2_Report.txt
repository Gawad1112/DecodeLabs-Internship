Task 2: Basic Encryption & Decryption

Project Overview:
In this task, I implemented a simple encryption and decryption program using the Caesar Cipher technique. The goal was to understand how data can be transformed from readable plaintext into unreadable ciphertext, then converted back to its original form through decryption.

Tools Used:
- Python 3
- Command Prompt / Terminal
- Caesar Cipher logic
- ASCII functions: ord() and chr()
- Modular arithmetic using % 26

Implementation Details:
The program asks the user to enter a message and a shift key. It then encrypts every alphabetic character by moving it forward in the alphabet based on the selected shift value. The program also keeps spaces, numbers, and punctuation unchanged so the output remains structured and readable.

For encryption, the logic used is:
E(x) = (x + n) % 26

For decryption, the logic is reversed:
D(x) = (x - n) % 26

The ord() function is used to convert characters into ASCII numbers, while chr() is used to convert the final numbers back into characters.

Results Achieved:
The program successfully displays:
- Original text
- Shift key
- Encrypted text
- Decrypted text
- Validation message confirming that decryption returned the original text

Sample Output:
Original Text : Hello Cyber Security 2026!
Shift Key     : 3
Encrypted Text: Khoor Fbehu Vhfxulwb 2026!
Decrypted Text: Hello Cyber Security 2026!
Validation: Successful. Decryption returned the original text.

Conclusion:
This task helped me understand the basic idea of confidentiality in cybersecurity. Even though the Caesar Cipher is not secure for modern systems, it is useful for learning the foundation of encryption, decryption, key-based shifting, ASCII conversion, and modular arithmetic.
